# @tool-developer/wx-utils
底层公共工具方法。

## 用法
```
npm install --save @tool-developer/wx-utils
```
or
```
yarn add @tool-developer/wx-utils
```

```
import utils from '@tool-developer/wx-utils'

```

## API
### .isEmpty(obj)
是否为空

### .isArray(val)
是否数组

### .isObject(val)
是否对象

### .isDate(val)
是否日期对象

### .merge(...args)
拷贝(合并)操作

### .deepMerge(...args)
深拷贝(合并)操作

### .forEach(obj, fn)
遍历处理
